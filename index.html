▸ ABOUT THIS PROJECT

PONG AI Training Lab is an educational project built to explore and demonstrate the core principles of Reinforcement Learning through a classic game.

The primary goal is learning by doing — by training AI agents in this environment, users gain intuitive understanding of how machines learn from experience without explicit programming.

CONCEPTS DEMONSTRATED

  • Q-Learning — a model-free RL algorithm where the agent learns the value of actions in states
  • Self-play — training by playing against oneself, a technique used in AlphaGo and AlphaZero
  • Exploration vs Exploitation — controlled by the epsilon (ε) parameter
  • Hyperparameter sensitivity — compare agents trained with different parameters
  • Elo-like rating — adjusted by training volume to reward skill over brute-force training

▸ USER GUIDE

STEP 1 — CREATE AN AGENT

Click + NEW AGENT in the left panel. Enter a name and optionally adjust the training parameters. Default values are good starting points. Click CREATE.

STEP 2 — TRAIN THE AGENT

Select the agent from the list, then click TRAIN. The agent will play thousands of simulated games against itself at high speed (no rendering). Progress is shown in the training bar. Training is complete when the bar reaches 100%.

STEP 3 — PLAY

Choose a Game Mode from the right panel, configure players in Match Setup, and click START.

KEYBOARD CONTROLS

  • W / S — Left player paddle (up / down)
  • ↑ / ↓ — Right player paddle (up / down)

TRAINING PARAMETERS

  • Episodes — Number of training games. More = stronger agent, slower training.
  • Learning Rate α — How fast the agent updates its knowledge (0.001–1.0). High = fast but unstable.
  • Discount γ — How much the agent values future rewards (0.5–0.999). High = long-term thinking.
  • Epsilon ε — Initial exploration rate. High = more random early moves.
  • ε Decay — How fast exploration decreases. Close to 1.0 = slow decay.
  • Ball Speed — Speed used during training. Match this to the match speed for best results.

RATING SYSTEM

Agents earn rating points when they win. The rating gain is adjusted by a training factor: a heavily trained agent earns fewer points when beating a lightly trained one. This prevents brute-force training from dominating the leaderboard.

GAME MODES

  • Human vs Agent — You play (W/S) against a trained AI.
  • Agent vs Agent — Watch two trained AIs compete.
  • Human vs Human — Two players on one keyboard.
  • Agent vs Self — Agent plays against an identical copy of itself.
